# daign-schedule

[![NPM package][npm]][npm-url]

Utilty mechanisms to manage the time-wise execution of functions.

## Installation

```sh
npm install @daign/schedule --save
```

## Scripts

#### Build

    npm run build

#### Run lint analysis

    npm run lint

#### Run unit tests with code coverage

    npm run test
